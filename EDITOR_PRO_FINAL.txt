  WORKFLOW EXEMPLO
═════════════════════════════════════════════════════════════════════════════

Criar uma Etiqueta de Produto:

1. Acesse /dashboard/products/manage
2. Clique aba "Templates"
3. Escolha template "Frutas - 40x60mm"
4. Clique "Editor Pro" → Editor abre

5. Adicione elementos:
   - Clique [T] → Arraste → Edite texto "Maçã Vermelha"
   - Clique [🖼] → Selecione imagem → Posicione
   - Clique [▨] → Configure QR Code → Posicione
   - Clique [▥] → Configure Barcode → Posicione

6. Organize no painel de camadas:
   - Reordene elementos
   - Clique Lock nas camadas finais
   - Use botões ↑↓ para Z-order

7. Salve:
   - Clique "Salvar" (azul)
   - Pronto! Layout persistido no Supabase
