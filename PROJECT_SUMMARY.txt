🚀 COMO USAR

1. Upload em Lote:
   Dashboard → Gerenciar Produtos → Upload em Lote
   Selecione seu CSV (name, ean obrigatórios)
   ✨ Produtos aparecem em "Meus Produtos"

2. Escolher Template:
   Dashboard → Gerenciar Produtos → Templates
   Filtrar por categoria (frutas, bebidas, etc)
   Clique "Usar" no template desejado
   ✨ Editor abre com template pré-configurado

3. Editor Visual:
   Selecione um produto
   Clique editar
   Use Konva.js para arrastar, redimensionar, rotacionar
   ✨ Mudanças salvam automaticamente

4. Tamanho Personalizado:
   Dashboard → Gerenciar Produtos → Tamanhos
   Digite nome, largura (mm), altura (mm)
   ✨ Novo tamanho disponível para uso
