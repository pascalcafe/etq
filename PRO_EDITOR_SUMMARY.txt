🚀 COMO ACESSAR
═══════════════════════════════════════════════════════════════════════════

Via Dashboard
  1. Acesse http://localhost:3000/dashboard/products/manage
  2. Clique na aba "Templates"
  3. Escolha um template (ex: Frutas - 40x60mm)
  4. Clique no botão "Editor Pro" (azul com ⚡)
  5. Pronto! Editor abre totalmente funcional

URL Direta
  http://localhost:3000/dashboard/labels/editor-pro

Com Parâmetros
  ?templateId=TEMPLATE_ID  → Carrega um template específico
  ?productId=PRODUCT_ID    → Associa a um produto

💡 EXEMPLOS DE USO
═══════════════════════════════════════════════════════════════════════════

Adicionar Texto:
  1. Clique no ícone [T] na toolbar
  2. Arraste no canvas
  3. Edite no painel direito: texto, size, peso

Adicionar Imagem:
  1. Clique no ícone [🖼]
  2. Selecione arquivo do PC
  3. Redimensione no canvas

Adicionar QR Code:
  1. Clique no ícone [▨]
  2. Defina o link no painel direito
  3. Posicione onde quiser

Salvar:
  1. Clique botão "Salvar" (azul)
  2. Aguarde confirmação
  3. Layout é persistido no Supabase
