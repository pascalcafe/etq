1. Preparar Arquivo
   • Excel/Google Sheets/LibreOffice
   • Colunas: Nome, Grupo, Valor, Unidade, Cód. Ref.
   • Salve como CSV (.csv) ou Excel (.xlsx)

2. Acessar Importador
   • Dashboard → Gerenciar Produtos
   • Aba "Itens de Cardápio" (dentro das abas)

   • Clique "Selecionar arquivo"
   • Escolha seu CSV/Excel

4. Revisar Preview
   • Sistema mostra primeiros 5 itens
   • Verifica formato
   • Erros aparecem em Alert

5. Enviar
   • Clique "Enviar"
   • Aguarde progresso
   • "X itens importados com sucesso!"

═════════════════════════════════════════════════════════════════════════════
  NÚMEROS ACEITOS
═════════════════════════════════════════════════════════════════════════════

Campo "Valor" - Todos aceitos:

  ✓ 45      → 45.00
  ✓ 45.00   → 45.00
  ✓ 45,00   → 45.00
  ✓ 8.50    → 8.50
  ✓ 8,5     → 8.50

Não funcionam:
  ✗ R$ 45.00    (remova símbolo)
  ✗ $ 45        (remova símbolo)
  ✗ 45 reais    (remova texto)
  ✗ "45,00"     (remova aspas)
  ✗ (vazio)     (campo obrigatório)
