  COLUNAS SUPORTADAS
═════════════════════════════════════════════════════════════════════════════

O arquivo deve ter exatamente 5 colunas (em qualquer ordem):

  [1] Nome ............ Texto | "Frango Grelhado"
  [2] Grupo ........... Texto | "Carnes"
  [3] Valor ........... Número | 45.00 ou 45,00
  [4] Unidade ......... Texto | "Kg", "L", "Porção", "Unidade"
  [5] Cód. Ref. ....... Texto | "FR-GRL-001"

  COMO USAR

Passo 1: Preparar Arquivo
  • Abra Excel/Google Sheets/LibreOffice
  • Crie colunas: Nome, Grupo, Valor, Unidade, Cód. Ref.
  • Preencha com seus dados
  • Salve como CSV (.csv) ou Excel (.xlsx)

  • Acesse: http://localhost:3000/dashboard/products/manage
  • Clique na aba "Itens de Cardápio" (4ª aba)

Passo 3: Selecionar Arquivo
  • Clique "Selecionar arquivo"
  • Escolha seu CSV/Excel

Passo 4: Revisar Preview
  • Sistema mostra os primeiros 5 itens
  • Verifica se formato está correto
  • Mensagens de erro aparecem aqui (se houver)

Passo 5: Enviar
  • Clique botão "Enviar"
  • Aguarde a barra de progresso completar
  • Sucesso! "X itens importados com sucesso!"

═════════════════════════════════════════════════════════════════════════════
  NÚMEROS ACEITOS
═════════════════════════════════════════════════════════════════════════════

Todos estes formatos são aceitos no campo "Valor":

  ✓ 45      → 45.00
  ✓ 45.00   → 45.00
  ✓ 45,00   → 45.00
  ✓ 45.5    → 45.50
  ✓ 45,5    → 45.50
  ✓ 8.50    → 8.50
  ✓ 8,50    → 8.50

Estes NÃO funcionam:
  ✗ R$ 45.00    (remova símbolo de moeda)
  ✗ $ 45.00     (remova símbolo de moeda)
  ✗ 45 reais    (remova texto)
  ✗ "45,00"     (remova aspas)
  ✗ (vazio)     (campo obrigatório)
